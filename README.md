# Pipeline e Análise de Dados — SCTEC Carreiras

Este projeto foi desenvolvido como parte prática do programa SCTEC Carreiras (Abril/2026), com o objetivo de simular um fluxo de dados de ponta a ponta: desde a extração e tratamento dos dados brutos até a disponibilização em um painel estratégico para tomada de decisão.

##  Tecnologias e Ferramentas Utilizadas
* **Python** (Google Colab)
* **Pandas & NumPy** (Limpeza e manipulação de dados)
* **GitHub** (Hospedagem do dataset e versionamento de código)
* **Looker Studio** (Construção do Dashboard e Visualização de Dados)

##  O Desafio e Solução
O projeto consistiu em receber uma base de dados bruta (Superstore) com diversas inconsistências, tipos de dados incorretos e valores nulos. 

Utilizando Python, realizei as seguintes etapas de Análise de Dados:
1. **Extração:** Conexão direta com a base de dados hospedada em nuvem.
2. **Tratamento (Tratamento de Encoding):** Correção de problemas de codificação (`latin1`) para leitura perfeita dos caracteres.
3. **Limpeza:** Identificação e tratamento de valores nulos, duplicados e inconsistências estruturais.
4. **Visualização:** Exportação dos dados limpos para modelagem e criação de relatórios visuais ricos no Looker Studio.

##  Estrutura do Repositório
* `Sample - Superstore.csv`: Dataset original utilizado no pipeline.
* *.ipynb (Arquivo do Notebook)*: Código completo e documentado com o passo a passo do tratamento dos dados.
